Username

Validation Class Description Feedback Class
.mr-username Specifies that the input value must be a valid username .mr-username-fb

myrules's username syntax

  • The username may use any of these ASCII characters:
    • uppercase and lowercase Latin letters A to Z and a to z
    • digits 0 to 9
    • some printable characters: -_.
  • it must start with uppercase or lowercase Latin letters and end with uppercase or lowercase Latin letters or digits
  • Printable characters cannot appear consecutively
  • You can change the printable characters

Applying in Input

Example: Applying in Input

Try to submit an invalid username.


Invalid username!

<form>
    <!--Validate username text field using .mr-username class"-->
    <input type="text" class="mr mr-username" placeholder="Username"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-username-fb">Invalid username!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Changing printable characters of username

the default printable characters of myrules's username are: -_. you can change them using any of the following supported printable characters: !#$%&'*+-/=?^_`{|}~.@

You can define your printable characters in the data-printablechars or printablechars attribute value

Example: changing printable characters

  • The username only accept these characters: alphanumerics (A-Za-z0-9) and these printable characters @._-$
  • it must start with uppercase or lowercase Latin letters and end with uppercase or lowercase Latin letters or digits
  • Printable characters cannot appear consecutively
  • Try to submit an invalid username

Invalid username!

<form>
    <!--Defining your printable characters in the data-printablechars attribute.-->
    <input type="text" class="mr mr-username" data-printablechars="@._-$" placeholder="Username"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-username-fb">Invalid username!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
