Pattern

Validation Attribute Description Feedback Class
pattern Specifies the value pattern of an input element .mr-pattern-fb

The pattern attribute can be used on this input types: text, date, search, url, tel, email, and password.

Applying in Input

Input Type Text Example

  • The Year input field can contain only four numbers (YYYY).
  • Try to input an invalid year.

Please input a valid year!

<form>    
    <!--Validate Year with pattern attribute-->
    <input type="text" class="mr" pattern="[0-9]{4}" placeholder="Year"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-pattern-fb">Please input a valid year!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Input Type Password Example

  • The password must contain 8 or more characters.
  • Try to input an invalid password.

The password must contain 6 or more characters!

<form>    
    <!--Validate Password with pattern attribute-->
    <input type="password" class="mr" pattern=".{6,}" placeholder="Password"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-pattern-fb">The password must contain 6 or more characters!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
