Multiple validation and feedbacks

You can validate an element with more than one Class or Attribute of Validation. All you need is set the Class and Attribute of Validation into the element.

<!--Attributes "type='email'" and "required" are Validation Attributes-->
<input type="email" class="mr" required placeholder="E-mail"> <br>
<!--Attributes "required" and "data-minlength" are Validation Attributes and class ".mr-password-good" is Validation Class-->
<input type="password" class="mr mr-password-good" required data-minlength="6" placeholder="Password"> <br>

The unique thing that you have to lean is how to display a multiple feedbacks.

<script src="http://exemplo.cdn/myrules.min.js"><script/>

Explaning the code

The class .mr indicates that the form element must be validated by myrules.

The attribute required is a validation attribute. The myrules classes and attributes of validation are classes and attributes that specifies the type of validation of the element. the validation attribute required specifies that the element value is required, it must be filled out before submitting the form. Learn more about myrules Classes and Attributes of Validation.

The class .mr-required-fb is an feedback class of validation attribute required. Feedbacks classes are classes used to display feedbacks of validations of the elements. The element with class .mr-required-fb will displayed when the required element was invalid.

The class .mr-feedback indicates the element where the feedbacks of validatiobs of element will placed.

For each element .mr needs one class .mr-feedback. The number of class .mr and class .mr-feedback must be iqual in the form.

Best loading experience

The feedbacks element are hidden after the page loaded. Depending the speed of page loading the user can see the feedbacks of validation before their hidden.

Proportionate better loading experience for the user adding the below CSS style sheet into the <head> section of your HTML page, to hide (display none) quick the feedbacks of validation preventing user see they.

.mr-feedback {
    display: none;
}

You can insert this style sheet by including the reference to the external style sheet file inside the <link> element or by internal style sheet, defining the style sheet within the <style> element.
