Accept

Validation Attribute Description Feedback Class
accept Specifies the accepted media types and file extensions of an input file. .mr-accept-fb

The accept attribute can only be used with input type file.

Attribute values

The accept attribute can have these values:

Value Description
file_extension File extension is a suffix of a filename. It consists of a period (.) followed by one or more characters, e.g: .png, .jpg, .gif, .doc, .html
image/* All image files are accepted
video/* All video files are accepted
audio/* All sound files are accepted
media_type A valid media type, with no parameters, e.g: audio/mpeg, application/pdf, image/png, font/otf

To specify more than one value, separate the values with a comma.

Applying in Input Type File

Example 1: Applying in Input Type File

  • The input accept all image files.
  • Try to submit an invalid image file.

Please input a valid image!

<form>    
    <!--Validate Input File with accept attribute-->
    <input type="file" class="mr" accept="image/*" placeholder="Files"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-accept-fb">Please input a valid image!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Example 2: Applying in Input Type File

  • The input accept all image files, PDF files and files with .mp3 extension.
  • Try to submit an invalid file.

Please input a valid file!

<form>    
    <!--Validate Input File with accept attribute-->
    <input type="file" class="mr" accept="image/*, application/pdf, .mp3" placeholder="Files"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-accept-fb">Please input a valid file!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
