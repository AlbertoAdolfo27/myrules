Match field value

Validation Class Description Feedback Class
.mr-match Specifies that the value of an input element with class .mr-match must match the value of other one correspondent input element with class .mr-match-subject .mr-match-fb

Applying in <input>

Input type="text" Example

Try to submit the form with different text fields values.



The nickname don't match!

<form>
    <!--Match two fields values using Validation Class ".mr-match" and ".mr-match-subject"-->
    <input type="text" class="mr-match-subject" placeholder="nickname"> <br>
    <input type="text" class="mr mr-match" placeholder="confirm the nickname"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-match-fb">The nickname don't match!</p>
    
    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Applying in <select>

Select Example

Try to submit the form with different options selected.






The options don't matches!

<form >
    <label>Select your favorite sport:</label> <br>
    <select class="mr-match-subject">  
        <option name="sport" value="football">Football</option>  
        <option name="sport" value="basketball">Basketball</option>
        <option name="sport" value="baseball">Baseball</option>  
        <option name="sport" value="golf">Golf</option> 
    </select>  <br><br>

    <label>Confirm your selection:</label> <br>    
    <select class="mr mr-match">  
        <option name="sport-confirm" value="football">Football</option>  
        <option name="sport-confirm" value="basketball">Basketball</option>
        <option name="sport-confirm" value="baseball">Baseball</option>  
        <option name="sport-confirm" value="golf">Golf</option> 
    </select>  <br>
    <p class="mr-feedback mr-match-fb">The options don't matches!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
