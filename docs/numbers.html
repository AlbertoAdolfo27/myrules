Numbers

Validation Attribute Validation Class Description Feedback Class
type="number" .mr-number Specifies that the input value must be a number. .mr-number-fb
.mr-number-int Specifies that the input value must be an integer number. .mr-number-int-fb

Numbers can be written with or without decimals and with scientific (exponent) notation.

Examples:

  • 23.15 - A number with decimals
  • 23 - A number without decimals
  • 19e7 - A number with exponent notation

To specifies that the input value must be a number add the type="number" attribute into the <input> element or .mr-number class.

To specifies that the input value must be an integer number (without decimals) add the .mr-number-int into the <input> element.

Applying in <input>

Input Example

  • The Quantity and Temperature text fields values must be a number.
  • The Age text field value must be an integer number.
  • The minimum allowed Age is 0. (lean how to validate minimum and maximum allowed value ).
  • Try to submit invalid values.

The Temperature must e a number!



The age must be an integer number!

<form>    
    <!--Validate Quantity with type="number" attribute-->
    <label>Quantity:</label> <br>
    <input type="number" class="mr" placeholder="Quantity"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-number-fb">The Quantity must e a number!</p>
    
    <!--Validate Temperature with .mr-number class-->
    <label>Temperature:</label> <br>
    <input type="text" class="mr mr-number" placeholder="Temperature"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-number-fb">The Temperature must e a number!</p>

    <!--Validate Age with .mr-number-int class and min attribute-->
    <label>Age:</label> <br>
    <input type="number" class="mr mr-number-int" min="0" placeholder="Age"> <br>
    <!--Validation feedback-->
    <div class="mr-feedback">
        <p class="mr-number-int-fb">The age must be an integer number!</p>
        <p class="mr-min-fb">The minimum allowed age is 0!</p>
    </div>
