Mincheck and Maxcheck

The Validation Attribute mincheck and maxcheck specifies the minimum and the maximum number of checkboxs allowed to select in a group of inputs type checkbox, respectively.

The Feedback Class of Validations Attributes mincheck is .mr-mincheck-fb and the Feedback Class of Validation Attribute maxcheck is .mr-maxcheck-fb.

The attributes mincheck and maxcheck are not attributes of HTML form elements, they're Custom Attributes.

The myrules HTML 5 Custom Attribute of mincheck is data-mincheck and myrules HTML 5 Custom Attribute of maxcheck is data-maxcheck. Use they if you want to make your HTML 5 page valid

The Validation Class mincheck donÂ´t specifies that the element requires a value. If your want to specifies that the element must be selected use the Validation Attribute required.

Elements with none value are ignored.

Applying in a group of <input type="checkbox">

Group of input type="checkbox" Example

  • The maximum allowed number of sports to select is 3

<form>

    <div class="mr mr-checkbox" data-mincheck="2" data-maxcheck="3">
        <label>Select your favorite sports:</label> <br>
        <input type="checkbox" name="sport1" value="football"> Football <br>  
        <input type="checkbox" name="sport2" value="basketball"> Basketball <br>  
        <input type="checkbox" name="sport4" value="golf"> Golf <br>
        <input type="checkbox" name="sport5" value="baseball"> Baseball <br>
        <input type="checkbox" name="sport6" value=""> None value <br>
    </div>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-mincheck-fb">Please, select at least 2 sports!</p>
        <p class="mr-maxcheck-fb">Please, select till 3 sports!</p>
    </div>
