Date

Validation Attribute Validation Class Description Feedback Class
type="date" .mr-date Specifies that the input value must be a valid complete date .mr-date-fb

You can validate a date using type="date" attribute on <input> element or .mr-date on <input type="text"> element.

Applying in Input Type Date

The date format of type="date" attribute is ISO 8601 (YYYY-MM-DD).

Depending on browser support, a date picker can show up in the input field.

Input Type Date Example

  • The valid date format is YYYY-MM-DD.
  • Try to input an invalid date.

Please input a valid date!

<form>    
    <!--Validate date with type="date" attribute-->
    <input type="date" class="mr" placeholder="Date"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-date-fb">Please input a valid date!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

You can also use the min and max attributes to add restrictions to dates. Learn now.

Applying in Input Type Text

To validate a date on an <input type="text"> use the .mr-date class.

The standard date format of .mr-class is ISO 8601 (YYYY-MM-DD). You can define other date input format.

Input Type Text Example

  • The valid date format is YYYY-MM-DD.
  • Try to input an invalid date.

Please input a valid date!

<form>    
    <!--Validate date with .mr-date class-->
    <input type="text" class="mr mr-date" placeholder="Date (YYYY-MM-DD)"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-date-fb">Please input a valid date!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Date formats

myrules provide 12 date input formats:

  • YYYY-MM-DD
  • YYYY/MM/DD
  • YYYY.MM.DD
  • YYYY,MM,DD
  • DD-MM-YYYY
  • DD/MM/YYYY
  • DD.MM.YYYY
  • DD,MM,YYYY
  • MM-DD-YYYY
  • MM/DD/YYYY
  • MM.DD.YYYY
  • MM,DD,YYYY

The standard date format is YYYY-MM-DD (ISO 8601 - The International Standard).

Use data-dateformat (or dateformat) attribute to specifies the date input format.

Define date format Example

  • The valid date format is MM/DD/YYYY
  • Try to input an invalid date.

Please input a valid date!

<form>    
    <!--Validate date with .mr-date class and data-dateformat attribute-->
    <input type="text" class="mr mr-date" data-dateformat="MM/DD/YYYY" placeholder="Date (MM/DD/YYYY)"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-date-fb">Please input a valid date!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
