imgwidth and imgheight

Validation Attribute Description Feedback Class
imgwidth or data-imgwidth Specifies the allowed width of an inputed image file, in pixels. .mr-imgwidth-fb
imgheight or data-imgheight Specifies the allowed height of an inputed image file, in pixels. .mr-imgheight-fb

The imgwidth (or data-imgwidth) and imgheight (or data-imgheight) attributes can only be used in inputed image files.

To specify more than one value, separate the values with a comma, e.g: data-imgwidth="300, 500"

Applying in Input Type File

Example: Applying in Input Type File

  • The inputed file must be an image (Lean how to specify accepted files).
  • The inputed image must have width of 450 pixels and height of 600 pixels.
  • Try to submit invalid an image.

Please input an image file!

The image must have width of 450 pixels!

The image must have height of 600 pixels!

<form>
    <!--Validate File with data-imgwitdh and data-imgheight attributes-->
    <input type="file" class="mr" accept="image/*" data-imgwidth="450" data-imgheight="600" placeholder="Image"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-accept-fb">Please input an image file!</p>
        <p class="mr-imgwidth-fb">The image must have width of 450 pixels!</p>
        <p class="mr-imgheight-fb">The image must have height of 600 pixels!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
