Scheme

The scheme is the protocol, it defines the type of Internet service. Examples of popular schemes include http, https, ftp, mailto, file, data and irc.

The scheme rules:

  • A scheme consisting of sequence of characters beginning with a letter and followed by any combination of letters, digits, plus (+), period (.), or hyphen (-).
  • Scheme component are followed by a colon (:)./li>

Defining accepted URL schemes

You can define the accepted schemes of the URL.

To define the accepted schemes set the scheme in the data-url-scheme or url-scheme attribute value.

For example, if you want to set hppt as the unique accepted scheme, set the value of attribute data-url-scheme or url-scheme "http". (data-url-scheme="http")

If you have more than one scheme them separete with comma, for example, data-url-scheme="http, https, file, ftp".

Remember, a scheme consisting of sequence of characters beginning with a letter and followed by any combination of letters, digits, plus (+), period (.), or hyphen (-).
