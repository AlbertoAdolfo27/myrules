ratio

Validation Attribute Description Feedback Class
ratio or data-ratio Specifies the aspect ratio of an inputed image file .mr-ratio-fb

The aspect ratio of an image is the proportional relationship between the width and the height of the image. It is expressed as two numbers separated by a colon (x:y), where x corresponds to the width and y to the height, e.g: 4:3, 3:2, 16:9, 1:1, 5:3, 5:4.

The ratio (or data-ratio) attribute can only be used in inputed image files.

To specify more than one value, separate the values with a comma, e.g: data-imgwidth="3:2, 16:9"

Applying in Input Type File

Example : Applying in Input Type File with multiple attribute

  • The inputed file must be an image (Lean how to specify accepted files).
  • The aspect ratio of the image must be 1:1
  • Try to submit invalid an image.

Please input an image file!

The aspect ratio of the image must be 1:1!

<form>
    <!--Validate File with data-ratio attribute-->
    <input type="file" class="mr" accept="image/*" data-ratio="1:1" placeholder="Image"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-accept-fb">Please input an image file!</p>
        <p class="mr-ratio-fb">The aspect ratio of the image must be 1:1!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
