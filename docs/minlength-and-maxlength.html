Minlength and Maxlength

Validation Attribute Description Feedback Class
minlength or data-minlength Specifies the minimum number of characters allowed in the filled out element (when the element value is not empty) .mr-minlength-fb
maxlength Specifies the maximum number of characters allowed in the element .mr-maxlength-fb

The Validation Attribute minlength donÂ´t specifies that the element requires a value. If your want to specifies that the element must be filled out use the Validation Attribute required.

Applying in <input>

Input Example

  • The minimum allowed number of characters of the input field Username after filled out is 4
  • The maximum allowed number of characters of the input field Username is 10
  • Try to submit an invalid value

Please, input at least 4 characters!

Please, input till 10 characters!

<form>
    <!--input text-->
    <input type="text" class="mr" data-minlength="4" maxlength="10" placeholder="Username"> <br>
    <!--Validation feedback-->
    <div class="mr-feedback">
        <p class="mr-minlength-fb">Please, input at least 4 characters!</p>
        <p class="mr-maxlength-fb">Please, input till 10 characters!</p>
    </div>


    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Applying in <textarea>

Textarea Example

  • The minimum allowed length of the Comment textarea after filled out is 5
  • The maximum allowed length of the Comment textarea is 20
  • Try to submit an invalid value


Please, input at least 5 characters!

Please, input till 20 characters!

<form>
    <label>Write a comment:</label> <br>
    <!--Textarea element-->    
    <textarea class="mr" name="comment" data-minlength="5" maxlength="20"></textarea>  <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-minlength-fb">Please, input at least 5 characters!</p>
        <p class="mr-maxlength-fb">Please, input till 20 characters!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
