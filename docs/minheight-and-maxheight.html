minheight and maxheight

Validation Attribute Description Feedback Class
minheight or data-minheight Specifies the minimum height of an inputed image file. .mr-minheight-fb
maxheight or data-maxheight Specifies the maximum height of an inputed image file. .mr-maxheight-fb

The minheight (or data-minheight) and maxheight (or data-maxheight) attributes can only be used in inputed image files.

Applying in Input Type File

Input Type File Example

  • The inputed file must be an image (Lean how to specify accepted files).
  • The minimum image height is 500 pixels
  • The maximum image height is 850 pixels
  • Try to submit invalid an image.

Please input an image file!

The minimum image height is 500 pixels!

The maximum image height is 850 pixels!

<form>
    <!--Validate File with data-minheight and data-maxheight attributes-->
    <input type="file" class="mr" accept="image/*" data-minheight="500" data-maxheight="850" placeholder="Image"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-accept-fb">Please input an image file!</p>
        <p class="mr-minheight-fb">The minimum image height is 500 pixels!</p>
        <p class="mr-maxheight-fb">The maximum image height is 850 pixels!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
