Required element's value

To Specify that the form element must be filled out (<input> elements) or selected (<select>) before submitting the form you add a .mr into the element's class then, add to the class .mr-required or attribute required.

Required feedback

To display the feedback validation when the required element has no value you must create a html element with the feedback text content and add the classes .mr-feedback and .mr-required-fb

Try to submit the form without fill out the text fields.


Please fill out this field


Please fill out this field

<form class="mr-colors">
    <!--Required text field-->
    <input type="text" placeholder="First name" class="mr" required> <br>
    <!--Validation feedback-->
    <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

    <!--Required text field-->
    <input type="text" placeholder="Last name" class="mr mr-required"> <br>
    <!--Validation feedback-->
    <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

    <input type="submit" value="submit">
</form>

Required checking

To Specify that the user must check a radio you must create a div with classes .mr, .mr-radio and attribute required or class .mr-required and add inside the created div the inputs type radio.

Try to submit the form without check a radio.

<form class="mr-colors">
    <p>Choice your gender</p>

    <div class="mr mr-radio mr-required">
        <input type="radio" name="gender" value="male"> Male <br>
        <input type="radio" name="gender" value="female"> Female <br>
    </div>
    <p class= "mr-feedback mr-required-fb">Please Choice your gender</p>

    <input type="submit" value="submit">
</form>

And to Specify that must checked at least one checkbox, create a div with classes .mr, .mr-checkbox and attribute required or class .mr-required and add inside the created div the inputs type checkbox.
