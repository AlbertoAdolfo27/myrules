Required element

To Specify that the element must be filled out before submitting the form you add a .mr into the element's class then, add to the class .mr-required or attribute required.

Required feedback

To display the feedback validation fo a required element you must create a html element with the feedback text content and add the classes .mr-feedback and .mr-required-fb

Example

Try to submit the form without fill out the text fields.


Please fill out this field


Please fill out this field

<form class="mr-colors">
    <!--Required text field-->
    <input type="text" placeholder="First name" class="mr" required> <br>
    <!--Validation feedback-->
    <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

    <!--Required text field-->
    <input type="text" placeholder="Last name" class="mr mr-required"> <br>
    <!--Validation feedback-->
    <p class= "mr-feedback mr-required-fb">Please fill out this field</p>

    <input type="submit" value="submit">
</form>
