Min and Max

Validation Attribute Description Feedback Class
min Specifies the minimum value of an input element .mr-min-fb
max Specifies the maximum value of an input element .mr-max-fb

Applying in <input type="number">

Input type="number" Example

  • The minimum quantity allowed is 1
  • The maximum quantity allowed is 100
  • Try to submit the invalid minimum and maximum value allowed

The minimum allowed quantity is 1!

The maximum allowed quantity is 100!

<form class="mr-colors">
    <!--Input type number-->
    <input type="number" class="mr" min="1" max="100" placeholder="Quantity"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-min-fb">The minimum allowed quantity is 1!</p>
        <p class="mr-max-fb">The maximum allowed quantity is 100!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Applying in <input type="text">

input type="text" with Validation Class .mr-date Example

To validate the min and max date in an <input type="text"> add the .mr-date class into the element class to specifies than the input value must be a date.

The standard date format of .mr-date Validation Class is ISO 8601 (YYYY-MM-DD). You can change it with dateformat Costum Attribute(or data-dateformat HTML 5 costum attribute).

Lean about the date formats of .mr-date Validation Class in Validate date subject.


Please input a valid date (YYYY-MM-DD)!

The minimum allowed date is 1941-10-25!

The maximum allowed date is 2015-10-25!

<form class="mr-colors">
    <!--Input type text with Validation Class ".mr-date"-->
    <input type="text" class="mr mr-date" min="" max="" placeholder="Birthday"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-date-fb">Please input a valid date (YYYY-MM-DD)!</p>
        <p class="mr-min-fb">The minimum allowed date is !</p>
        <p class="mr-max-fb">The maximum allowed date is !</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

min="today" and max="today" date

Set the current date as value of the min or max attributes by using the value "today". It will update every day the value of the min or max attributes to the actual date.

  • The minimum allowed date in the Meeting date text field is (current date)
  • The maximum allowed date in the Birthday text field is (current date)
  • Try to submit the invalid minimum and maximum allowed date in the text fields


Please input a valid date (YYYY-MM-DD)!

The minimum allowed date is current date!




Please input a valid date (YYYY-MM-DD)!

The maximum allowed date is current date!

<form class="mr-colors">
    <label>Meeting date:</label> <br> 
    <!--Input type date-->
    <input type="date" class="mr" min="today" placeholder="Meeting date"> <br>
    
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-date-fb">Please input a valid date (YYYY-MM-DD)!</p>
        <p class="mr-min-fb">The minimum allowed date is current date!</p>
    </div>

    
    <label>Birthday:</label> <br> 
    <!--Input type date-->
    <input type="date" class="mr" max="today" placeholder="Birthday"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-date-fb">Please input a valid date (YYYY-MM-DD)!</p>
        <p class="mr-max-fb">The maximum allowed date is current date!</p>
    </div>
