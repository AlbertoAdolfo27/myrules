Multi-validate an element

You can validate an element with more than one Class or Attribute of Validation. All you need is to set the Class and Attribute of Validation into the element.

<input type="text" class="mr mr-latin-space" required data-minlength="2">
<!--Attributes "data-minlength" and "required" are Validation Attributes and class ".mr-latin-space" is Validation Class-->

To display the feedback of each validation, add the .mr-feedback class into a <DIV> element, then add child element of each Feedback validation of the element.

Example 1: Validate the First Name and a Last Name text fields with more than one type of validation

  • The First Name and Last Name are requireds fields
  • The text fields must macth Latim with space ([a-zA-ZÀ-ÖØ-öø-ʯ ])
  • The text fields must have at least two characters
  • Try to input an invalid First Name and a Last Name

<form>
    <!--First Name text field-->
    <input type="text" class="mr mr-latin-space" required data-minlength="2" placeholder="First name"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-required-fb">Please fill out this text field!</p>
        <p class="mr-latin-space-fb">Please input a valid name!</p>
        <p class="mr-minlength-fb">A valid name must have at least two characters!</p>
    </div>

    <!--Last Name text field-->
    <input type="text" class="mr mr-latin-space" required data-minlength="2" placeholder="Last name"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-required-fb">Please fill out this text field!</p>
        <p class="mr-latin-space-fb">Please input a valid name!</p>
        <p class="mr-minlength-fb">A valid name must have at least two characters!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
