Group of input type="checkbox" Example

To specifies that in a group of input radio must be selected an option (with value), use a <DIV> element with classes .mr, .mr-radio classes and Class or Attribute of Validation of required element (.mr-required class or required attribute) and append input type radio elements.

The required attribute is conceived to use in <input>, <select> and <textarea> elements, not in <DIV>, then, if you want a valid HTML 5 page prefer to use the .mr-required Validation Class.

Try to submit the form without select any radio option.

Female

<form class="mr-colors">
    <!--Group of radio inputs-->
    <div class="mr mr-radio mr-required">
        <label>Select your gender:</label> <br> 
        <input type="radio" name="gender" value="Male"> Male <br>  
        <input type="radio" name="gender" value="Female"> Female <br>
    </div>
    <!--Validation feedbacks-->
    <p class="mr-feedback mr-required-fb">please select one option!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Group of input type="checkbox" Example

To specifies that in a group of checkbox inputs must be checked at least one option with value, add a <DIV> element with classes .mr and .mr-checkbox and Class or Attribute of Validation of requerid element and append the input checkbox elements.

Try to submit the form without check any checkbox option.


Football
Basketball
Golf

please check at least one option!

<form class="mr-colors">
    <!--Group of checkbox inputs-->
    <div class="mr mr-checkbox mr-required">
        <label>Select your favorite sports:</label> <br>
        <input type="checkbox" name="sport1" value="football"> Football <br>  
        <input type="checkbox" name="sport2" value="basketball"> Basketball <br>  
        <input type="checkbox" name="sport4" value="golf"> Golf <br>
    </div>
    <!--Validation feedbacks-->
    <p class="mr-feedback mr-required-fb">please check at least one option!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
