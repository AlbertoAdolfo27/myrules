Minsize and Maxsize

Validation Attribute Description Feedback Class
minsize or data-minsize Specifies the mimimum size of an input file. .mr-minsize-fb
maxsize or data-maxsize Specifies the maximum size of an input file. .mr-maxsize-fb

The minsize (or data-minsize) and maxsize (or data-maxsize) attributes can only be used with input type file.

Attribute values

The data-minsize and data-maxsize attributes can have these values:

Value Description
n n unit of byte, e.g: data-minsize="1048576", data-maxsize="2097152".
nkB n unit of kilobyte, e.g: data-minsize="600kB", data-maxsize="500kB".
nMB n unit of megabyte, e.g: data-minsize="50MB", data-maxsize="150MB".
nGB n unit of gigabyte, e.g: data-minsize="1GB", data-maxsize="1GB".

Applying in Input Type File

Input Type File Example

  • The minimum allowed file size is 300 kilobytes.
  • The maximum allowed file size is 5 megabytes.
  • Try to submit an file with invalid allowed file size.

The minimum allowed file size is 300 kilobites!

The maximum allowed file size is 5 megabytes!

<form>
    <!--Validate File with data-minsize and data-maxsize attributes-->
    <input type="file" class="mr" minsize="300kb" maxsize="5mb" placeholder="File"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-minsize-fb">The minimum allowed file size is 300 kilobites!</p>
        <p class="mr-maxsize-fb">The maximum allowed file size is 5 megabytes!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Input Type File with multiple attribute Example

  • You can enter more than one file.
  • The minimum allowed file size of each file is 1 megabyte.
  • The maximum allowed file size of each file is 5 megabytes.
  • Try to submit an file with invalid allowed file size.

The minimum allowed size of each file is 1 megabyte!

The maximum allowed size of each file is 5 megabytes!

<form>
    <!--Validate File with data-minsize and data-maxsize attributes-->
    <input type="file" class="mr" multiple minsize="1mb" maxsize="5mb" placeholder="File"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-minsize-fb">The minimum allowed file size of each file is 1 megabyte!</p>
        <p class="mr-maxsize-fb">The maximum allowed file size of each file is 5 megabytes!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Sum the size of all entered files

If you want to determine the maximum allowed size of the sum of all entered files just add the .mr-sum-sizes class into the input element class.

Input Type File with multiple attribute exemple 2

  • You can enter more than one file.
  • The maximum allowed size of the sum of all file is 7 megabytes.
  • Try to submit an file with invalid allowed file size.

The maximum allowed size of the sum of all file is 7 megabytes!

<form>
    <!--Validate File with data-maxsize attribute and .mr-sum-sizes class-->
    <input type="file" class="mr mr-sum-sizes" multiple maxsize="7mb" placeholder="File"> <br>
    <!--Validation feedbacks-->
    <p class="mr-feedback mr-maxsize-fb">The maximum allowed size of the sum of all file is 7 megabytes!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
