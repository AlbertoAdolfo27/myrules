minwidth and maxwidth

Validation Attribute Description Feedback Class
minwidth or data-minwidth Specifies the minimum width of an inputed image file, in pixels. .mr-minwidth-fb
maxwidth or data-maxwidth Specifies the maximum width of an inputed image file, in pixels. .mr-maxwidth-fb

The minwidth (or data-minwidth) and maxwidth (or data-maxwidth) attributes can only be used in inputed image files.

Applying in Input Type File

Example: Applying in Input Type File

  • The inputed file must be an image (Lean how to specify accepted files).
  • The minimum image width is 400 pixels
  • The maximum image width is 2500 pixels
  • Try to submit invalid an image.

Please input an image file!

The minimum image width is 400 pixels!

The maximum image width is 2500 pixels!

<form>
    <!--Validate File with data-minwidth and data-maxwidth attributes-->
    <input type="file" class="mr" accept="image/*" data-minwidth="400" data-maxwidth="2500" placeholder="Image"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-accept-fb">Please input an image file!</p>
        <p class="mr-minwidth-fb">The minimum image width is 400 pixels!</p>
        <p class="mr-maxwidth-fb">The maximum image width is 2500 pixels!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
