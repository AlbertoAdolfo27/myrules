Minselect and Maxselect

Validation Attribute Description Feedback Class
minselect or data-minselect Specifies the minimum allowed number of selection on a select element or an input type file with attribute multiple. .mr-minselect-fb
maxselect or data-maxselect Specifies the maximum allowed number of selection of a select element or an input type file with attribute multiple. .mr-maxselect-fb

The Validation Class minselect donÂ´t specifies that the element requires a value. If your want to specifies that the element must be selected use the Validation Attribute required.

Elements with none value are ignored.

Applying in <select>

Select Example

  • The minimum allowed number of sports to select is 2
  • The maximum allowed number of sports to select is 3
  • Try to select less than 2 and more than 3 sports


Please, select at least 2 sports!

Please, select till 3 sports!

<form>
    <label>Select your favorite sport:</label> <br>
    <!--Select element-->    
    <select class="mr" multiple data-minselect="2" data-maxselect="3"> 
        <option name="sport" value="cyclism">Cyclism</option> 
        <option name="sport" value="football">Football</option>  
        <option name="sport" value="basketball">Basketball</option>  
        <option name="sport" value="baseball">Baseball</option>  
        <option name="sport" value="golf">Golf</option> 
    </select>  <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-minselect-fb">Please, select at least 2 sports!</p>
        <p class="mr-maxselect-fb">Please, select till 3 sports!</p>
    </div>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>

Applying in <input type="file">

Input type="file" Example

  • The minimum allowed number of files to select is 2
  • The maximum allowed number of files to select is 3
  • Try to select less than 2 and more than 3 files

Please, select at least 2 files!

Please, select till 3 files!

<form>
    <!--input text-->
    <input type="file" class="mr" multiple data-minselect="2" maxselect="3" placeholder="Select files"> <br>
    <!--Validation feedbacks-->
    <div class="mr-feedback">
        <p class="mr-minselect-fb">Please, select at least 2 files!</p>
        <p class="mr-maxselect-fb">Please, select till 3 files!</p>
    </div>


    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
