Quick start

Copy-paste the following <script> near the end of your web pages, before the closing </body> tag. If you need, download the source file and host yourself.

<script src="https://cdn.jsdelivr.net/gh/AlbertoAdolfo27/myrules/dist/myrules.js"></script>

Basic validation

Start with this basic validation example that shows the validation of requireds text fields. Try to submit the form without fill out the text fields.

  • The inputs text First name and Last name are required fields
  • Try to submit the form without fill out the text fields

Please fill out this field


Please fill out this field

<!doctype html>
<html lang="en">
    <head>
        <title>Hello, world!</title>
        <meta charset="utf-8">
    </head>
    <body>
        <form>
            <!--Required text field-->
            <input type="text" placeholder="First name" class="mr" required> <br>
            <!--Validation feedback-->
            <p class="mr-feedback mr-required-fb">Please fill out this field</p>

            <!--Required text field-->
            <input type="text" placeholder="Last name" class="mr" required> <br>
            <!--Validation feedback-->
            <p class="mr-feedback mr-required-fb">Please fill out this field</p>

            <input type="submit" value="submit">
            <input type="reset" value="reset">
        </form>

        <!--myrules-->
        <script src="https://cdn.jsdelivr.net/gh/AlbertoAdolfo27/myrules/dist/myrules.js"></script>
    </body>
</html>

Explaning the code

.mr - class that indicates that the form element must be validated by myrules.

required - is an attribute validation that specifies that the element value is required, it must be filled out before submitting the form. myrules uses classes and attributes to validate the form elements, learn more about myrules Classes and Attributes of Validation.

.mr-required-fb is the feedback class of validation attribute required. Feedbacks classes are classes used to display feedbacks of validations of the elements. The element with class .mr-required-fb will displayed when the required element was invalid.

The class .mr-feedback indicates the element where the feedbacks validation elements will placed.

For each element .mr needs one class .mr-feedback. The number of class .mr and class .mr-feedback must be iqual in the form element.

Hide feedbacks faster when page loading

The feedbacks element are hidden after the page loaded. Depending the internet speed the user can see the feedbacks of validation before their hidden.

Add the below CSS style sheet into the <head> section of your HTML page, to hide (display none) quick the feedbacks validation preventing user see they.

.mr-feedback {
    display: none;
}

You can insert this style sheet by including the reference to the external style sheet file inside the <link> element or by internal style sheet, defining the style sheet within the <style> element.
