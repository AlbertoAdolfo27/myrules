Step

Validation Attribute Description Feedback Class
step Specifies the legal number intervals of an input element. .mr-step-fb

Applying in <input>

input Example

  • The legal number intervals of Quantity text fields is 5.
  • Try to submit invalid number.

Please input a legal number!

<form>    
    <!--Validate Quantity with step attribute-->
    <input type="number" class="mr" step="5" placeholder="Quantity"> <br>
    <!--Validation feedback-->
    <p class="mr-feedback mr-step-fb">Please input a legal number!</p>

    <input type="submit" value="submit">
    <input type="reset" value="reset">
</form>
